# Example Usage Commands

This file contains various example commands you can try with the AI Coding Assistant.

## Basic Questions

```bash
# Get help with Python basics
ca ask "How do I create a virtual environment in Python?"

# Get help with specific language features
ca ask "Explain list comprehensions in Python with examples" --language python

# Get help with debugging
ca ask "My code is throwing a KeyError, how do I fix it?" --language python
```

## Code Analysis Examples

```bash
# Analyze a Python file
ca analyze my_script.py

# Analyze a JavaScript file
ca analyze app.js

# Analyze any code file
ca analyze complex_algorithm.cpp
```

## Code Generation Examples

```bash
# Generate a simple function
ca generate "Create a function to check if a number is prime"

# Generate language-specific code
ca generate "REST API endpoint for user login" --language python

# Generate and save to file
ca generate "Binary search tree implementation" --language python --output bst.py
```

## Advanced Examples

```bash
# Get help with complex topics
ca ask "How to implement a singleton pattern?" --language python --context "Thread-safe implementation"

# Get optimization advice
ca ask "How to optimize database queries?" --context "Working with PostgreSQL and Django ORM"

# Get architecture advice
ca ask "Best practices for microservices communication" --context "Python Flask applications"
```

## Specific Problem Solving

```bash
# Debug specific errors
ca ask "How to fix 'ImportError: No module named' error?" --language python

# Performance optimization
ca ask "How to make this sorting algorithm faster?" --context "Working with large datasets"

# Security concerns
ca ask "How to prevent SQL injection in my queries?" --language python --context "Using PostgreSQL"
```

## Learning New Technologies

```bash
# Learn new frameworks
ca ask "How to get started with FastAPI?" --language python

# Compare technologies
ca ask "Difference between REST and GraphQL APIs"

# Best practices
ca ask "Python coding best practices for large projects" --language python
```

Try these examples and modify them based on your specific needs!
